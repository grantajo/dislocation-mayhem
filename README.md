
# Dislocation Mayhem

Dislocation Mayhem 1 is a simple game where you are a dislocation that is trying
to get away from defects that are coming toward you.  

Your goal is to get out of the material.

## Dislocation Mayhem 2 Ideas

There are a bunch of 'infected' dislocations that are increasing and causing your material to get really brittle and start breaking off pieces. Kind of like a plague.  

You move through the material, grain by grain. Each has a different composition, temperature, etc. that changes the way the fight works.

- This is a turn-based strategy game where you can use materials science
to get away from defects in a material
- Stress meter per region
- Grain boundaries are walls and you move through different grains for different levels? Each grain has it's own thing like higher temp, high dislocation density, etc.

- Enemies
  - Dislocations
    - Dislocations glide along specific slip planes
    - Dislocations can climb via vacancy diffusion
    - Frank-Read source generators
    - Edge, screw, and mixed dislocations move differently?
  - Twins
- Player
  - Can create 'friendly' dislocations
  - Can locally change the temperature
  - Dislocation annihilation
  - Power ups
    - Alloying Elements (Mn increases strain hardening, etc.)
    - Substitutional for lattice distortion
    - Interstitial for local hardening (reducing enemy speed)
    - Induce twinning to change slip systems
